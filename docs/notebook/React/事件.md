### 函数相关

## 事件

为input绑定onChange事件
```
constructor(){
    super();
    this.state = {
        inputValue : ''
    }
}

handleClick = (e) => {
    this.setState({
        inputValue : e.currentTarget.value
    })
}

render() {
    return (
            <input 
                type="text"
                value={this.state.inputValue}
                onChange={this.handleClick}
            />
    )
}
```

>为数组添加数据时要用**concat()**返回新数组

```
this.state = {
    todos:[{
        id: 1,
        title: '吃饭',
        isCompleted: true
    },{
        id: 2,
        title: '睡觉',
        isCompleted: false
    }]
}

addTodo = (todoTitle) => {
    this.setState({
        //先数据操作再赋值给todos，push()返回数组个数
        todos:this.state.todos.concat({  
            id:3,
            title:todoTitle,
            isCompleted: true
        })
    },() => {
        console.log(this.state.todos)
    })
}
```
