## 盒子模型
```
盒子模型就是指CSS布局中的每一个元素，在浏览器的解释中，都被当作一个盒状物。
对我们来说，只需要理解元素在页面中所占据的位置。
```
```
元素最终所占宽度：
左边框宽 + 左内边距 + 内容宽度 + 右内边距 + 右边框宽
```


```
width:50%;      相对值，相对于父元素的宽度，进行百分比计算
width: 200px;   固定值，固定的大小，不会跟随页面大小改变而改变
height: 50%;
height: 200px;
```
```
padding: 50px 40px 30px 20px;    上50px 右40px 下30px 左20px，顺时针方向，设置四个方向内边距
padding: 50px 40px 30px;    上50px 左右40px 下30px
padding: 50px 40px；  上下50px 左右40
padding: 50px;    内边距padding：控制边框以内的空隙大小，具体是指内容到边框之间的空位,四个方位都是50px的内边距

margin:30px 20px 10px 5px  上50px 右40px 下30px 左20px，顺时针方向，设置四个方向外边距
margin: 30px 20px 10px;   上30px 左右20px 下10px
margin: 30px 20px;   上下30px 左右20px
margin: 30px;    上下左右 30px
```
### 注意
```
margin 外边距
水平方向，相邻元素的margin是叠加的，
垂直方向，相邻的元素的margin是合并的（取大值为准）
```
