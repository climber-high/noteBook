### JS基础

## break与continue的区别


| break | continue |
| ----- | ----- |
| 立即终止循环 | 终止当前循环 |

##### 例
```
var i=0
while(i<10){
	i++;
	if(i==5){
		break;/*立即终止循环*/
		continue;//终止的当前循环
	}
	console.log(i);
}
```
