## HTML简介
```
HTML 全称是Hypertext Markup Language————超文本标记语言 Hypertext是指超文本语言(文本 + 图片 + 
视频 + 音频 + 链接)； 请注意HTML是专门写给浏览器浏览的语言；
```
### HTML语法
```
HTML的层次结构
<!DOCTYPE html>
<html>
     <head>
         <meta charset="utf8">
         <title>我的第一个网站</title>
     </head>
     <body>
                  欢迎来到HTML5的世界
	 </body>
</html>
```

### 标签
```
HTML元素由开始标签和结束标签组成
位于开始标签和结束标签中的文本 是元素的内容。
HTML 标签有开始必须有结束。如果是没有内容的标签（空标签），用 />来结束。
标签名推荐用小写。
标签具有属性， 属性用来表示标签的性质和特征。属性要在开始标签中指定
```
