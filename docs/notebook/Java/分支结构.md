### 分支结构

## 顺序结构
>从上往下逐行执行，每句必走

## 分支结构
>有条件的执行某语句，并非每句必走
	1.if结构
	
	2.if...else结构
	
	3.if...else if 结构
	
	4.switch...case结构
		优点:效率高，结构清晰
		缺点:整数，相等
		
		int num=2;
		switch (num) {  //byte,short,int,char
			case 1:  //相当于if(num==1)
				System.out.println(111); 
				break;
			case 2:  //以此处为入口
				System.out.println(222);
				break;
			case 3:
				System.out.println(333);
				break;
			default:
				System.out.println(444);
		}-
		所有case都没有执行再执行default;
